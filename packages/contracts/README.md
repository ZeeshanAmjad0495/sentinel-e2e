# @sentinel/contracts

The dependency-free contract layer of [Sentinel](https://github.com/ZeeshanAmjad0495/sentinel-e2e). It declares the shared, framework-agnostic types — locators, actions, assertions, sessions, capabilities, and the telemetry vocabulary — that every driver and the AI analyzer implement against. It carries no runtime dependencies so any package can depend on it without pulling in a driver.

## Install

```sh
npm install @sentinel/contracts
```

See the [root README](https://github.com/ZeeshanAmjad0495/sentinel-e2e#readme) for the full Sentinel overview.
