# @sentinel/driver-selenium

The Selenium WebDriver driver for [Sentinel](https://github.com/ZeeshanAmjad0495/sentinel-e2e). It implements the `@sentinel/contracts` driver/session interface on top of `selenium-webdriver`, compiling Sentinel locator strategies to CSS/XPath selectors and emitting `@sentinel/core` telemetry for every action and assertion. `selenium-webdriver` ships as a direct dependency.

## Install

```sh
npm install @sentinel/driver-selenium
```

See the [root README](https://github.com/ZeeshanAmjad0495/sentinel-e2e#readme) for the full Sentinel overview.
